プライバシーポリシー

最終更新日: 2025年8月15日
重要: 本アプリケーションは、ユーザーの個人情報を一切収集いたしません。

1. 情報の収集について

本アプリケーションは、以下の情報を収集いたしません：

  • 個人を特定できる情報（氏名、住所、電話番号、メールアドレスなど）
  • 端末情報（IMEI、広告ID、位置情報など）
  • 利用状況やアクセス履歴
  • その他あらゆる個人情報

2. データの保存について

本アプリケーションは、ユーザーのデータを外部サーバーに送信したり、保存したりすることはありません。 アプリの動作に必要な設定情報などは、お客様の端末内でのみ保存され、外部に送信されることはありません。

3. 第三者への情報提供

本アプリケーションは情報を収集していないため、第三者に情報を提供することはありません。

4. アクセス解析ツールについて

本アプリケーションでは、Google Analyticsなどのアクセス解析ツールは使用しておりません。

5. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更や当アプリケーションの機能追加に伴い、予告なく変更される場合があります。 変更後のプライバシーポリシーは、本ページに掲載された時点で効力を生じるものとします。
